ParamTypeDescription
optionsObjectcontains defaults and overrides
options.urlStringthe url of the remote deploy.sh service
options.applicationStringthe deployed application name to alter
options.mongoStringthe mongo connection string used by deploy.sh service when running deploy serve --mongo ''
